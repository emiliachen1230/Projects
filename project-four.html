Rondo Door Bell
Interaction Device

Product x PCB

Category

Interactive product / musical device / prototype

Keywords

Arduino, Bluetooth, sound interaction, object interface

Concept

把門鈴從單一訊號變成一種可感知、可互動、可傳遞情緒的聲音裝置。

Overview

這頁就是你原本希望點圖後進去的作品詳情頁版本。之後你可以把圖片、文案、技術細節換成自己真正的內容。

Rondo 是一個以音樂互動為核心的裝置。它把門鈴這種日常物件重新設計成可被感知、可被演奏的介面，讓「有人到來」不只是提示，而是帶有情緒、節奏與身體性的事件。

裝置整合感測、按鍵、聲音輸出與電子電路，透過簡單但明確的互動邏輯，讓使用者能夠在進入空間的那一刻接觸到聲音。它不是單純播放聲音，而是把聲音變成一種關係媒介。
