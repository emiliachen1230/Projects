# Personal Portfolio Site

這是一個可直接部署到 GitHub Pages 的靜態網站。

## 使用方式

1. 把 `index.html` 放到你的 GitHub repository 根目錄
2. 到 GitHub repository 的 `Settings` → `Pages`
3. `Build and deployment` 選 `Deploy from a branch`
4. Branch 選 `main`，資料夾選 `/(root)`
5. 儲存後等幾分鐘，網站就會上線

## 你需要先改的地方

請打開 `index.html`，把以下內容換成你自己的：

- `YOUR NAME`
- `Artist / Engineer / Researcher`
- Email
- GitHub / Instagram 連結
- About / CV / Project 文字
- 如果要放真實圖片，可以把 `.project-cover` 改成 `<img>`，或是改背景圖

## 建議

如果你之後要：
- 加作品內頁
- 加真實圖片
- 加多語切換
- 加深色模式
- 拆成 `index.html + style.css + script.js`

再往下整理會更好維護。
